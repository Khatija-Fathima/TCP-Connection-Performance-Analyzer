# TCP Connection Performance Analyzer

## Overview
This project analyzes TCP connection performance using Python Socket Programming, Mininet, and Wireshark.

## Features
- Persistent TCP Connections
- New Connection Analysis
- Throughput Measurement
- Response Time Analysis
- Wireshark Packet Inspection
- Mininet Network Emulation

## Technologies Used
- Python
- Mininet
- Wireshark
- Linux Networking
